# Carpool App — UX Research

UX research documentation for a carpooling solution designed to help parents coordinate school runs and extracurricular rides more efficiently.

This repository contains the full research presentation, including:
- problem framing
- user research findings
- personas and journey insights
- information architecture exploration
- card sorting analysis
- key conclusions and opportunity areas
